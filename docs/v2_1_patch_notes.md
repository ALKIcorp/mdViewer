# MD Viewer V2.1 Patch Notes (Proposed)

## New Features
- [ ] **Local Storage Autosave**: Automatically save editor content to the browser's local storage so work is not lost on refresh.
- [ ] **Theme Toggle**: Add a button to toggle between Dark Mode (default) and Light Mode.
- [ ] **Word & Character Count**: Add a status bar at the bottom of the editor showing real-time statistics.
- [ ] **New Block Types**:
    - **Image Block**: Insert images via URL.
    - **Table Block**: Insert a basic markdown table structure.

## Improvements
- [ ] **Export Filename**: Allow users to specify a custom filename when exporting (currently defaults to `document`).
- [ ] **Mobile Optimization**: Improve the Block Panel layout on smaller screens.

## Bug Fixes
- [ ] Fix any lingering cursor jump issues during complex formatting operations.
